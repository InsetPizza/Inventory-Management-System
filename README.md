# Inventory-Management-System
The Inventory Management System is a digital tool designed for a small business to replace manual tracking methods like spreadsheets or notebooks. At its core, it's a software application, powered by Python and a MySQL database, that provides a central place to monitor goods, manage stock levels, track sales, and oversee suppliers.

# Prerequisites
Install Python 3.x.

Install MySQL Server.

Install dependencies: pip install -r requirements.txt

# How to Run
Open your MySQL client and ensure the server is running.

Open db.py and update the password field with your MySQL password.

Run the dashboard: python dashboard.py
